PART B: HYPOTHESIS TESTING

Hypothesis Testing Results

5 Hypotheses Validated via Statistical Modeling

H₁: CES Difference Between States

SUPPORTED

Independent Samples t-test: t(398) = 12.41, p < 0.001

H₂: Communication → Education

SUPPORTED

Binary Logistic Regression: Nagelkerke R² = 0.31

H₃: Communication → Healthcare

PARTIALLY SUPPORTED

Two-Way ANOVA: Interaction F(3,389) = 5.14, p = 0.002

H₄: Communication → Environment

SUPPORTED

Chi-square: χ²(3) = 24.88, Cramer's V = 0.25

H₅: Communication → Livelihood

SUPPORTED

Multiple Linear Regression: β(CES) = 0.67, R² = 0.50
